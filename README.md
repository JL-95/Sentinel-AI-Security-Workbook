# Sentinel AI Security Workbook

## English Introduction
This repository contains an Azure Sentinel workbook package for monitoring AI-related security posture, usage, authorization changes, and operational risk.

### Included Files
- `AI Workbook.json`  English version
- `AI Workbook - cn.json`  Chinese version

### What This Workbook Covers
- AI threat and attack monitoring
- API security and anomaly visibility
- AI service configuration change tracking
- AI service access authorization monitoring
- AI application usage and access tracking
- Incident response efficiency indicators

### How to Use
1. Open Microsoft Sentinel in Azure.
2. Go to **Workbooks**.
3. Import either workbook JSON file from this repository.
4. Select your Log Analytics workspace and time range.
